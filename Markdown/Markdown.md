# 마크다운 문법 정리

## 1.헤더

    큰제목 (======)
    작은제목 (------)
    #  H1
    ##  H2
    ###  H3
    ####  H4
    #####  H5
    ######  H6

## 2.순서있는 목록

    1. 첫번째
    2. 두번째
    3. 세번째

결과
1. 첫번째
2. 두번째
3. 세번째

## 3.순서없는 목록

    * 첫번째
      * 두번째
        * 세번째
    
결과

 * 첫번째
  * 두번째
  * 세번째
